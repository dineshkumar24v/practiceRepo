Heading tags in html :

Heading tags in HTML are used to define headings or titles for sections of content on a webpage. They range from h1 to h6, with h1 being the largest and most important heading, and h6 being the smallest and least important.

HTML tags are classified into :

1. Paired Tags (or) Container tags :

Paired tags have both opening and closing tags, and they enclose content between them. For example, p and p in the above example.

2. Self-Closing Tags (or) Empty tags (or) Void tags :

Self-closing tags do not have a separate closing tag, and they are used to insert elements that do not contain any content or do not require closing. For example, img for an image element, or br for a line break.

Types of Elements :

1. Block Level Element :

The tag which occupies whole width and creates a new line is called a Block level element

Eg: h1, h6, p, div, semantic elements

2. Inline-Block Element :

The elements which occupies only required space and doesnot create a new line and we can apply width, height, margin and padding properties.

Eg: img, iframe, input, button etc

3. Inline Element :

The elements which occupies only required space and doesnot create a new line, in css we cannot apply width, height, padding top and bottom and margin top and bottom.

Eg: span, formating tags, b, strong, i, etc
